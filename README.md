Mini Projet Blog :
Application de blog réalisée avec React et Vite dans le cadre de ma formation à la 3W Academy.
Fonctionnalités

Affichage des articles du blog
Création d'un nouvel article

Technologies utilisées :

React,
Vite,
JavaScript,
CSS.
